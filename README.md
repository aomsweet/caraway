# caraway

An asynchronous network proxy based on `netty`. Support `HTTP`/`SOCKS4`/`SOCKS5` proxy protocol on one port.

## Features

- [x] HTTP
- [x] HTTP Tunnel
- [ ] HTTP MITM
- [ ] SOCKS4
- [ ] SOCKS5
- [ ] Username/password authentication
- [ ] GSSAPI authentication
- [ ] Chain proxies
- [ ] Http interceptor
- [ ] Rate monitor
